# Unit Converter

A simple unit converter web application built with vanilla HTML, CSS, and JavaScript.

![Doodle Style](./style/border.svg)

## Features

- **Length Conversion** — Milimeter, Centimeter, Meter, Kilometer, Inch, Foot, Yard, Mile
- **Weight Conversion** — Miligram, Gram, Kilogram, Ounce, Pound
- **Temperature Conversion** — Celcius, Fahrenheit, Kelvin

## How It Works

1. Select a conversion type from the navigation (Length, Weight, or Temperature)
2. Enter a value and choose the units to convert from and to
3. Click **Convert** to see the result
4. Click **Reset** to start a new conversion

## Tech Stack

- HTML5
- CSS3 ([DoodleCSS](https://github.com/chr15m/DoodleCSS))
- Vanilla JavaScript

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/unit-converter.git
```
2. Open `index.html` in your browser
